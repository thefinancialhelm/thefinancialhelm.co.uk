Brand Palette & Usage Guide

Core Brand Colours

⚠️ Important Accessibility Guidelines

The original Brand Gold fails WCAG AA contrast requirements when used as text on white backgrounds. Use the darker text variants below for all text applications to ensure accessibility compliance.

Brand Navy

#05192A | HSL(207, 68%, 9%)

Our primary brand colour representing trust, stability, and professional expertise. The nautical connection reinforces our "helm" metaphor.

Use for:

  • Main headings and navigation
  • Dark background sections
  • Logo on light backgrounds
  • Professional documents

Brand Gold

#CA9E5D | HSL(37, 45%, 58%)

Our accent colour for backgrounds, buttons, and icons. Important: Do not use for text on white backgrounds (fails WCAG contrast).

Use for:

  • Primary button backgrounds
  • Icon colours and decorative elements
  • Logo accent elements
  • Background highlights

Gold Text (Large)

#8B6F3F | HSL(37, 45%, 35%)

Accessible darker gold for headlines and large text (24px+). Meets WCAG AA contrast requirements on white backgrounds.

Use for:

  • Main headlines (24px+)
  • Section titles
  • Navigation hover states
  • Large accent text

Gold Text (Body)

#7A6633 | HSL(37, 50%, 30%)

Darkest gold variant for body text and smaller elements (16px+). Provides excellent contrast and readability.

Use for:

  • Body text accents (16px+)
  • Link colours
  • Important inline text
  • Medium-sized highlights

Light Gold

#D9B57F | HSL(37, 45%, 72%)

A softer variant of our gold for subtle accents and hover states. Provides warmth without being overwhelming.

Use for:

  • Button hover states
  • Subtle background accents
  • Secondary highlights
  • Gradient combinations

Brand Cream

#F5F1E8 | HSL(37, 40%, 92%)

A warm, approachable neutral that softens the professional palette. Creates comfortable reading environments.

Use for:

  • Content area backgrounds
  • Card and panel backgrounds
  • Alternative to pure white
  • Warm, welcoming sections

WCAG Accessibility Compliance

Text Size Guidelines

24px+ (Headlines):
Use Gold Text variant

16px+ (Body text):
Use Accessible Gold variant

14px- (Small text):
Use Navy text only

Contrast Ratios

Gold Text: 4.8:1 ✓

Accessible Gold: 6.2:1 ✓

Navy Text: 12.6:1 ✓

Original Gold: 2.47:1 ✗

Legal Requirement

All public-facing websites must meet WCAG 2.1 AA standards for accessibility compliance. This is not optional - it's a legal requirement in many jurisdictions.

Do's and Don'ts

✓ DO

  • Use navy and gold as primary brand colours
  • Use darker gold variants for text accessibility
  • Test all colour combinations for WCAG AA compliance
  • Use original gold for backgrounds and buttons only
  • Use Gold Text variant for headlines (24px+)
  • Use Accessible Gold for body text (16px+)
  • Use navy text for small text (14px and below)
  • Maintain sufficient contrast ratios always

✗ DON'T

  • Never use original gold as text on white backgrounds
  • Never use any gold variant for text smaller than 16px
  • Don't ignore WCAG accessibility requirements
  • Don't use gold text without checking contrast ratios
  • Don't create colour combinations not in this guide
  • Don't use more than 3 colours in a single design
  • Don't modify the brand colours without approval
  • Don't assume colour combinations are accessible
